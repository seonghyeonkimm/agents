# Acceptance Criteria: PRO Expert Mode Optimization

## 기능 1: 최적화 추천 컬럼 표시

- [ ] 전문가모드 캠페인 목록에 '광고 최적화 추천' 컬럼이 표시된다
- [ ] 각 캠페인 행에 추천 상태가 표시된다 (추천 있음/없음/진단중)
- [ ] 추천이 있는 캠페인은 컬럼에 시각적 지시자(아이콘/배지)가 표시된다
- [ ] 사용자가 컬럼의 항목을 클릭하면 추천 상세 정보가 표시된다

## 기능 2: 진단 및 추천 생성

- [ ] 각 캠페인의 설정, 예산 소진율, 성과 지표가 분석되어 진단이 수행된다
- [ ] 진단 결과 우선순위가 높은 액션 아이템이 생성된다
- [ ] 추천은 캠페인별로 개별적으로 표시된다
- [ ] 같은 캠페인에 여러 추천이 있으면 우선순위 순서로 정렬되어 표시된다

## 기능 3: 추천 액션 실행

- [ ] 추천 항목을 클릭하면 해당 설정 화면으로 이동한다
- [ ] 일부 추천은 "즉시 적용" 버튼으로 원클릭 적용이 가능하다
- [ ] 사용자가 추천을 적용하면 성공 메시지가 2초 동안 표시된다
- [ ] 적용된 추천의 상태가 업데이트되어 표시된다 (완료/미완료)

## 기능 4: 사용자 경험

- [ ] 기존 캠페인 목록 레이아웃은 유지되고 새 컬럼만 추가된다
- [ ] 추천 기능은 선택적으로 숨길 수 있다 (컬럼 숨김 옵션)
- [ ] 대시보드 로드 시 추천 데이터는 별도로 비동기 로드된다
- [ ] 페이지 이동 없이 추천 정보가 실시간으로 갱신된다
