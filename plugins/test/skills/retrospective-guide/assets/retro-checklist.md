# Retrospective Facilitator Checklist

## Before (1 day ahead)

- [ ] Choose retrospective format
- [ ] Book meeting room / set up video call
- [ ] Send calendar invite with agenda
- [ ] Prepare digital board (Miro, FigJam, etc.) or physical materials
- [ ] Review previous retro's action items
- [ ] Optional: Send pre-retro survey

## Setup (15 min before)

- [ ] Test tech (screen share, audio, video)
- [ ] Open retrospective board/template
- [ ] Prepare timer
- [ ] Have backup format ready

## During the Retrospective

### Opening (5 min)
- [ ] Welcome everyone
- [ ] State the Prime Directive
- [ ] Explain the format
- [ ] Set ground rules

### Review (5 min)
- [ ] Go through previous action items
- [ ] Celebrate completed items
- [ ] Discuss blockers on incomplete items

### Gather Data (15-20 min)
- [ ] Give silent writing time (5 min)
- [ ] Share and discuss each category
- [ ] Group similar items

### Generate Insights (10 min)
- [ ] Identify patterns and themes
- [ ] Discuss root causes
- [ ] Dot vote on priorities

### Decide Actions (10 min)
- [ ] Create specific action items
- [ ] Assign owners
- [ ] Set due dates

### Close (5 min)
- [ ] Summarize key takeaways
- [ ] Thank participants
- [ ] Schedule next retrospective

## After

- [ ] Share retro summary within 24 hours
- [ ] Add action items to team backlog
- [ ] Follow up on action items weekly
