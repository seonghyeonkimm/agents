## 0. 필수 파일 확인 (CRITICAL)

먼저 프로젝트 루트에서 필수 파일들이 존재하는지 확인합니다:

```bash
# 필수 파일 존재 여부 확인
ls -la init.sh feature-list.json 2>&1
```

**필수 파일이 없는 경우 즉시 중단:**

`init.sh` 또는 `feature-list.json` 파일이 없으면, 다음 메시지를 출력하고 **즉시 중단**하세요:

---

**프로젝트 설정이 필요합니다**

필수 파일이 누락되었습니다:
- [ ] init.sh
- [ ] feature-list.json

먼저 `/ai-workflow:setup` 명령어를 실행하여 프로젝트를 설정해주세요.

---

**중요: 필수 파일이 없으면 아래 단계를 절대 진행하지 마세요.**

## 1. 초기화 스크립트 실행

`init.sh` 파일을 실행합니다:
- 파일 내용을 먼저 확인하여 안전한지 검토
- 실행 권한이 있는지 확인 후 실행
- 실행 결과를 사용자에게 보고

## 2. Feature List 파악

`feature-list.json` 파일을 읽어 작업 목록을 파악합니다:
- JSON 내용을 파싱하여 작업 항목들을 정리
- 각 feature의 상태(완료/진행중/대기)를 파악
- 우선순위가 있다면 정리

## 3. 진행 상황 컨텍스트 보충

`claude-progress.txt` 파일을 읽어 최근 작업 맥락을 파악합니다:

```bash
# claude-progress.txt 파일 찾기
ls -la claude-progress.txt 2>&1
```

파일을 찾으면:
- 최근 완료된 작업들을 요약
- 현재 진행 중인 작업 파악
- 다음에 해야 할 작업 식별

파일이 없으면:
- 진행 기록이 아직 없음을 알리고 계속 진행

## 4. 요약 보고

모든 단계를 완료한 후, 다음 형식으로 요약을 제공합니다:

---

**프로젝트 초기화 완료**

**작업 목록 (feature-list.json):**
- 총 N개 feature
- 완료: N개
- 진행중: N개
- 대기: N개

**최근 진행 상황 (claude-progress.txt):**
- [최근 작업 요약]
- [현재 진행 중인 작업]

**다음 추천 작업:**
- [우선순위에 따른 다음 작업 제안]
